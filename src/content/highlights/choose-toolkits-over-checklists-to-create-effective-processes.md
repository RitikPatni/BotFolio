## Highlights

- An exhaustive checklist implicitly says to the reader, “I don’t trust you to get this right, so I need you to prove that you did it properly by checking each of these boxes in order.”
  5
  A toolkit inverts that relationship. It says, “You’re the expert here, so we trust you to get this done. Here are all the tools you might need to get there.”
