Hello Everyone,

Hope everyone had a nice week. Mine was a quiet one thanks to all the scorching heat, which made doing anything almost impossible. With that, let's hope for nicer temps and let's start with this week's stuff.

## 📚 To Read

[The Message Behind the Medium of a Personal Blog](https://blog.jim-nielsen.com/2022/the-message-and-medium-of-the-personal-blog/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

When I read this I thought: if “the medium is the message”, what’s the message behind the medium of a personal blog?

[Things I Wish I’d Known Earlier In My Career](https://www.smashingmagazine.com/2022/06/things-to-know-earlier-in-your-career/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

This is a very enjoyable and useful read. So many useful tips for early career folks.

[How to Reverse an Animated GIF](https://davidwalsh.name/how-to-reverse-an-animated-gif?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

This is a very useful and fun to use tip.

[The Demo → Demo Loop](https://daverupert.com/2022/06/demo-to-demo-loop/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

The secret to making successful creative products.

[Single Element Loaders: The Bars](https://css-tricks.com/single-element-loaders-the-bars/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

This is again very useful. Next one and in the same series that I shared last week [Single Element Loaders: The Bars](https://css-tricks.com/single-element-loaders-the-spinner/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

[Style Queries](https://una.im/style-queries/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26)

You may have heard of [container queries](https://css-tricks.com/next-gen-css-container/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26) and the new [contain-level-3 spec](https://drafts.csswg.org/css-contain-3/?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26) which is currently in [experimental browsers](https://caniuse.com/css-container-queries?utm_source=ritik&utm_medium=email&utm_campaign=reading-list-26), but have you heard of style container queries, which are also a part of this (very exciting) spec?
